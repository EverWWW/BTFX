# BTFX 软件打包发布计划

## 1. 概述

### 1.1 目标
为 **BTFX 步态智能分析系统** 创建完整的打包发布流程，生成可在 Windows 平台上一键安装的 EXE 安装包。

### 1.2 当前项目信息
| 项目 | 值 |
|------|-----|
| 项目名称 | BTFX |
| 显示名称 | 步态智能分析系统 |
| 目标框架 | .NET 10.0-windows |
| 项目类型 | WPF 桌面应用 |
| 当前版本 | V1.0.0.1 |

### 1.3 依赖项目
- ToolHelper.Communication
- ToolHelper.Database
- ToolHelper.DataProcessing
- ToolHelper.LoggingDiagnostics

## 2. 打包方案选择

### 2.1 可选方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **Inno Setup** | 免费、轻量、自定义强、社区成熟 | 需要编写脚本 | ⭐⭐⭐⭐⭐ |
| **WiX Toolset** | MSI 标准格式、企业级 | 配置复杂、学习曲线陡 | ⭐⭐⭐ |
| **NSIS** | 免费、脚本灵活 | 脚本语法较老 | ⭐⭐⭐ |
| **MSIX** | Windows 10+ 原生支持 | 仅限 Win10+、需签名 | ⭐⭐⭐ |
| **ClickOnce** | VS 集成、自动更新 | 灵活性差、企业环境受限 | ⭐⭐ |
| **单文件发布** | 无需安装程序 | 无安装体验、无快捷方式 | ⭐⭐ |

### 2.2 推荐方案：Inno Setup
**理由**：
1. 完全免费且开源
2. 生成单个 EXE 安装包，用户体验好
3. 支持自定义安装界面、多语言
4. 可创建桌面快捷方式、开始菜单项
5. 支持卸载程序自动生成
6. 社区成熟，文档丰富

## 3. 实施步骤

### 3.1 阶段一：项目配置优化
**目标**：优化 BTFX.csproj，为发布做准备

1. **添加版本信息属性**
   ```xml
   <PropertyGroup>
     <Version>1.0.0.1</Version>
     <FileVersion>1.0.0.1</FileVersion>
     <AssemblyVersion>1.0.0.1</AssemblyVersion>
     <Company>BTFX Team</Company>
     <Product>步态智能分析系统</Product>
     <Copyright>Copyright © 2024-2026</Copyright>
     <Description>步态智能分析系统 - Gait Intelligent Analysis System</Description>
   </PropertyGroup>
   ```

2. **配置发布参数**
   ```xml
   <PropertyGroup Condition="'$(Configuration)'=='Release'">
     <PublishSingleFile>false</PublishSingleFile>
     <SelfContained>true</SelfContained>
     <RuntimeIdentifier>win-x64</RuntimeIdentifier>
     <PublishReadyToRun>true</PublishReadyToRun>
     <IncludeNativeLibrariesForSelfExtract>true</IncludeNativeLibrariesForSelfExtract>
   </PropertyGroup>
   ```

3. **添加应用程序图标**
   ```xml
   <PropertyGroup>
     <ApplicationIcon>Resources\Images\Logo\app.ico</ApplicationIcon>
   </PropertyGroup>
   ```

### 3.2 阶段二：创建发布配置文件
**目标**：创建 pubxml 发布配置

1. **创建发布配置文件** `BTFX\Properties\PublishProfiles\FolderProfile.pubxml`
   - 目标运行时：win-x64
   - 部署模式：自包含（Self-contained）
   - 输出目录：`publish\win-x64`

### 3.3 阶段三：准备 Inno Setup 脚本
**目标**：创建安装包构建脚本

1. **创建目录结构**
   ```
   BTFX\
   └── Installer\
       ├── BTFX.iss              # Inno Setup 脚本
       ├── license.txt           # 许可协议
       ├── readme.txt            # 自述文件
       └── Assets\
           ├── installer.ico     # 安装包图标
           ├── header.bmp        # 安装向导头部图片
           └── sidebar.bmp       # 安装向导侧边栏图片
   ```

2. **Inno Setup 脚本功能**
   - 安装路径选择
   - 桌面快捷方式（可选）
   - 开始菜单快捷方式
   - 自动创建数据目录
   - 卸载程序
   - 版本检测与覆盖安装

### 3.4 阶段四：创建构建脚本
**目标**：一键构建安装包

1. **PowerShell 构建脚本** `BTFX\Installer\build-installer.ps1`
   ```powershell
   # 1. 清理发布目录
   # 2. 执行 dotnet publish
   # 3. 复制额外文件（许可证、自述文件等）
   # 4. 调用 Inno Setup 编译器生成安装包
   # 5. 输出安装包到 Output 目录
   ```

2. **批处理脚本** `BTFX\Installer\build-installer.bat`
   - 简化调用，适合 CI/CD

### 3.5 阶段五：文档与测试
**目标**：确保打包流程可靠

1. **创建打包说明文档** `BTFX\Installer\README.md`
   - 环境要求
   - 打包步骤
   - 常见问题

2. **测试清单**
   - [ ] 全新安装测试
   - [ ] 覆盖安装测试
   - [ ] 卸载测试
   - [ ] 快捷方式测试

## 4. 安装包功能规格

- **开始菜单**：BTFX / 步态智能分析系统
- **开始菜单**：BTFX / 卸载

### 4.4 卸载功能
- 删除程序文件
- 删除快捷方式
- 可选：保留/删除用户数据
