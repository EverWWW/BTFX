# UI规范与组件设计

**目标**: 定义统一的UI规范、自定义控件和样式，确保界面一致性

---

## 1. 设计规范

### 1.1 Material Design 主题

| 项目 | 设置 |
|-----|------|
| 基础主题 | Light（默认）/ Dark |
| 主色调(Primary) | Blue (#2196F3) |
| 副色调(Secondary) | LightBlue (#03A9F4) |
| 字体 | 阿里巴巴普惠体 (Alibaba PuHuiTi) |

### 1.2 颜色规范

| 用途 | 颜色 | 说明 |
|-----|------|------|
| 主色 | #2196F3 (Blue) | 主要按钮、标题栏、导航高亮 |
| 副色 | #03A9F4 (LightBlue) | 次要强调 |
| 成功 | #4CAF50 (Green) | 成功提示、已连接状态 |
| 警告 | #FF9800 (Orange) | 警告提示、连接中状态 |
| 错误 | #F44336 (Red) | 错误提示、未连接状态 |
| 文字主色 | #212121 | 主要文字 |
| 文字次色 | #757575 | 次要文字、提示文字 |
| 分割线 | #E0E0E0 | 分割线、边框 |
| 背景色 | #FAFAFA | 页面背景 |
| 卡片背景 | #FFFFFF | 卡片、对话框背景 |

### 1.3 字体规范

| 类型 | 大小 | 粗细 | 用途 |
|-----|------|------|------|
| Headline5 | 24px | Bold | 页面标题 |
| Headline6 | 20px | Medium | 卡片标题、对话框标题 |
| Subtitle1 | 16px | Medium | 列表项标题 |
| Body1 | 14px | Regular | 正文内容 |
| Body2 | 14px | Regular | 次要内容 |
| Caption | 12px | Regular | 提示文字、标签 |
| Button | 14px | Medium | 按钮文字 |

### 1.4 间距规范

| 类型 | 值 | 用途 |
|-----|-----|------|
| 页面边距 | 24px | 页面内容与边缘 |
| 卡片内边距 | 16px | 卡片内容与边缘 |
| 组件间距 | 16px | 组件之间 |
| 紧凑间距 | 8px | 紧凑布局 |
| 按钮间距 | 8px | 按钮之间 |

### 1.5 圆角规范

| 类型 | 值 |
|-----|-----|
| 按钮 | 4px |
| 卡片 | 4px |
| 对话框 | 4px |
| 输入框 | 4px |
| 小圆角 | 2px |

### 1.6 阴影规范

使用 Material Design 标准阴影等级：
- Dp2: 轻微阴影（输入框聚焦）
- Dp4: 卡片默认阴影
- Dp8: 悬浮卡片、下拉菜单
- Dp16: 对话框

## 2. 控件样式规范

### 2.1 按钮样式

| 类型 | 样式 | 用途 |
|-----|------|------|
| 主要按钮 | MaterialDesignRaisedButton | 主要操作（登录、保存、确认） |
| 次要按钮 | MaterialDesignOutlinedButton | 次要操作（取消、重置） |
| 文字按钮 | MaterialDesignFlatButton | 链接类操作 |
| 图标按钮 | MaterialDesignIconButton | 工具栏操作 |
| 强调按钮 | MaterialDesignRaisedAccentButton | 特别强调操作 |

### 2.2 输入框样式

| 类型 | 特性 |
|-----|------|
| 标准输入框 | 浮动标签、清空按钮 |
| 密码输入框 | 浮动标签、显隐切换按钮 |
| 数字输入框 | 仅允许数字、范围限制 |
| 搜索框 | 搜索图标、清空按钮 |
| 多行输入框 | 自动高度或固定高度 |

### 2.3 下拉框样式

| 特性 | 说明 |
|-----|------|
| 浮动标签 | 选中后标签上浮 |
| 占位提示 | 未选中时显示提示 |
| 清空按钮 | 可选，允许清空选择 |

### 2.4 日期选择器

| 特性 | 说明 |
|-----|------|
| 格式 | yyyy-MM-dd |
| 限制 | 不可选未来日期 |
| 最早日期 | 当前年份 - 150年 |

### 2.5 时间显示格式

| 场景 | 格式 | 示例 |
|-----|------|------|
| 顶部状态栏时间 | yyyy-MM-dd dddd HH:mm:ss | 2026-01-08 星期三 14:30:25 |
| 列表中的日期时间 | yyyy-MM-dd HH:mm | 2026-01-08 14:30 |
| 仅日期 | yyyy-MM-dd | 2026-01-08 |
| 报告日期 | yyyy年MM月dd日 | 2026年01月08日 |

### 2.6 数据表格 (DataGrid)

| 特性 | 说明 |
|-----|------|
| 行高 | 48px |
| 表头高度 | 56px |
| 斑马纹 | 奇偶行交替背景色 |
| 选中高亮 | 选中行背景色变化 |
| 悬停效果 | 鼠标悬停行背景色变化 |
| 虚拟化 | 启用虚拟化提升性能 |

## 3. 自定义控件设计

### 3.1 CustomTitleBar 自定义标题栏

| 属性 | 类型 | 说明 |
|-----|------|------|
| Title | string | 标题文字 |
| ShowLogo | bool | 是否显示Logo |
| ShowVersion | bool | 是否显示版本号 |

| 子元素 | 说明 |
|-------|------|
| Logo区域 | 左侧Logo图片 |
| 标题区域 | 应用名称 |
| 版本区域 | 版本号（可点击） |
| 按钮区域 | 最小化、全屏/还原、关闭 |

| 功能 | 说明 |
|-----|------|
| 拖拽移动 | 支持鼠标拖拽移动窗口 |
| 双击全屏 | 双击标题栏切换全屏 |

### 3.2 PaginationControl 分页控件

| 属性 | 类型 | 说明 |
|-----|------|------|
| CurrentPage | int | 当前页码 |
| TotalPages | int | 总页数 |
| TotalRecords | int | 总记录数 |
| PageSize | int | 每页条数 |
| PreviousPageCommand | ICommand | 上一页命令 |
| NextPageCommand | ICommand | 下一页命令 |
| GoToPageCommand | ICommand | 跳转命令 |

| 显示内容 | 格式 |
|---------|------|
| 中文 | "第 X 页 / 共 Y 页，共 Z 条记录" |
| 英文 | "Page X of Y, Z records total" |

| 按钮 | 说明 |
|-----|------|
| 上一页 | 第一页时禁用 |
| 下一页 | 最后一页时禁用 |
| 页码输入 | 可输入页码跳转 |

### 3.3 NumericUpDown 数字输入框

| 属性 | 类型 | 说明 |
|-----|------|------|
| Value | double | 当前值 |
| Minimum | double | 最小值 |
| Maximum | double | 最大值 |
| Increment | double | 步进值 |
| DecimalPlaces | int | 小数位数 |
| IsReadOnly | bool | 是否只读 |

| 功能 | 说明 |
|-----|------|
| 增减按钮 | 点击增加/减少值 |
| 直接输入 | 允许键盘输入 |
| 范围校验 | 超出范围自动修正到边界值 |
| 格式化 | 按小数位数格式化显示 |

### 3.4 SearchBox 搜索框

| 属性 | 类型 | 说明 |
|-----|------|------|
| Text | string | 搜索文本 |
| Placeholder | string | 占位提示 |
| MaxLength | int | 最大长度 |
| SearchCommand | ICommand | 搜索命令 |

| 功能 | 说明 |
|-----|------|
| 搜索图标 | 左侧显示搜索图标 |
| 清空按钮 | 有内容时显示清空按钮 |
| 回车搜索 | Enter键触发搜索 |

### 3.5 StatusIndicator 状态指示器

| 属性 | 类型 | 说明 |
|-----|------|------|
| Status | enum | 状态枚举 |
| StatusText | string | 状态文字 |

| 显示 | 说明 |
|-----|------|
| 指示灯 | 圆形指示灯，颜色随状态变化 |
| 文字 | 状态描述文字 |

| 状态 | 颜色 |
|-----|------|
| Connected | Green |
| Connecting | Orange |
| Disconnected | Gray |
| Failed | Red |

## 4. 对话框设计规范

### 4.1 对话框通用规范

| 项目 | 规范 |
|-----|------|
| 宽度 | 根据内容，最小320px，最大800px |
| 圆角 | 4px |
| 阴影 | Dp16 |
| 标题 | 左上角，Headline6字体 |
| 关闭按钮 | 右上角X按钮 |
| 按钮区域 | 底部右对齐，主要按钮在右 |

### 4.2 确认对话框 (ConfirmDialog)

| 内容 | 说明 |
|-----|------|
| 图标 | 可选，警告/提问/错误图标 |
| 标题 | 对话框标题 |
| 消息 | 确认信息内容 |
| 按钮 | "是" / "否" 或 "确定" / "取消" |

### 4.3 编辑对话框规范

| 项目 | 规范 |
|-----|------|
| 表单布局 | 垂直排列，每个字段一行 |
| 标签位置 | 使用浮动标签 |
| 必填标识 | 浮动标签后加 * |
| 验证提示 | 字段下方显示红色错误信息 |
| 按钮 | "保存" / "取消" |

## 5. 行为 (Behaviors)

### 5.1 PasswordBindingBehavior

用于 PasswordBox 的双向绑定（WPF PasswordBox 不支持直接绑定）。

| 附加属性 | 类型 | 说明 |
|---------|------|------|
| Password | string | 绑定密码值 |

### 5.2 NumericOnlyBehavior

限制 TextBox 只能输入数字。

| 附加属性 | 类型 | 说明 |
|---------|------|------|
| IsNumericOnly | bool | 是否仅数字 |
| AllowDecimal | bool | 是否允许小数 |
| AllowNegative | bool | 是否允许负数 |

### 5.3 TextLengthLimitBehavior

文本长度限制，超出部分截断。

| 附加属性 | 类型 | 说明 |
|---------|------|------|
| MaxLength | int | 最大长度 |

### 5.4 WindowDragBehavior

窗口拖拽移动行为。

| 附加属性 | 类型 | 说明 |
|---------|------|------|
| IsDraggable | bool | 是否可拖拽 |

| InverseBoolToVisibilityConverter | bool → Visibility (True=Collapsed) |

| RoleToStringConverter | UserRole枚举 → 角色名称 |
| EnumToDescriptionConverter | 枚举 → Description特性值 |
| StatusToColorConverter | 状态枚举 → Brush颜色 |

## 8. 响应式布局指南

### 窗口尺寸断点

| 断点 | 宽度 | 说明 |
|-----|------|------|
| 小屏 | < 1280px | 最小支持尺寸 |
| 中屏 | 1280px - 1920px | 标准尺寸 |
| 大屏 | > 1920px | 大屏/全屏 |

### 布局策略

| 策略 | 说明 |
|-----|------|
| Grid | 主要布局使用Grid |
| 比例 | 使用 * 比例分配空间 |
| 最小宽度 | 设置 MinWidth 防止过度压缩 |
| 滚动 | 内容超出时使用 ScrollViewer |

## 9. 无障碍设计

### 基本要求

| 项目 | 要求 |
|-----|------|
| Tab顺序 | 合理的Tab键导航顺序 |
| 快捷键 | 主要功能支持快捷键 |
| 提示文字 | 所有按钮设置 ToolTip |
| 颜色对比 | 文字与背景有足够对比度 |

### 快捷键规划

| 快捷键 | 功能 |
|-------|------|
| F11 | 全屏切换 |
| ESC | 退出全屏/关闭对话框 |
| Ctrl+S | 保存 |
| Ctrl+N | 新增 |
| Ctrl+F | 搜索聚焦 |
| Enter | 确认/搜索 |
