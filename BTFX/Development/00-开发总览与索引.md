# BTFX 步态智能分析系统 - 开发总览与索引

## 项目概述

### 项目名称
**BTFX** - 步态智能分析系统 (Gait Intelligent Analysis System)

### 系统功能
连接摄像机采集患者行走视频，进行关节点识别和步态参数分析，输出分析报告。

### 技术栈
| 技术 | 说明 |
|-----|------|
| **框架** | WPF (.NET 10.0) |
| **架构** | MVVM (原生实现，不使用Prism) |
| **MVVM工具** | CommunityToolkit.Mvvm (ObservableObject、RelayCommand) |
| **依赖注入** | Microsoft.Extensions.DependencyInjection |
| **UI设计** | Material Design In XAML Toolkit (最新稳定版，蓝色调) |
| **数据库** | SQLite (通过 ToolHelper.Database 的 SqlSugar ORM) |
| **图表库** | OxyPlot.Wpf |
| **已集成工具库** | ToolHelper.Communication、ToolHelper.Database、ToolHelper.DataProcessing、ToolHelper.LoggingDiagnostics |

### 核心特性
- ? 单窗体多视图切换架构
- ? Material Design UI风格，无边框自定义标题栏
- ? 多语言支持 (中文/英文) - 实时切换
- ? 主题颜色切换 - 实时生效
- ? 全屏显示 (ESC键或按钮退出)
- ? 三级用户权限管理 (管理员/操作员/游客)
- ? 患者信息管理 (含导入导出CSV/Excel)
- ? 测量数据管理 (步态视频与参数) - *待算法完成*
- ? 报告生成与管理 (A4纸张) - *待细化*
- ? 数据备份与恢复 (手动/自动定时)
- ? 防止重复启动
- ? 在线/离线激活机制

### 重要约束
> ?? **测量模块**: 当前算法功能未完成，测量界面暂时保留空白框架，优先开发软件框架和其他通用功能

> ?? **开发规范**: 严格遵循 `DevelopmentSpecifications.md` 中的所有要求

## 架构设计决策

### MVVM 实现方案

| 组件 | 选型 | 说明 |
|-----|------|------|
| **ViewModel基类** | CommunityToolkit.Mvvm 的 ObservableObject | 自动实现 INotifyPropertyChanged |
| **命令** | CommunityToolkit.Mvvm 的 RelayCommand | 支持异步命令和CanExecute |
| **依赖注入** | Microsoft.Extensions.DependencyInjection | 与 ToolHelper 库统一 |
| **导航服务** | 自定义 INavigationService | 使用 ContentControl 实现视图切换 |
| **视图关联** | DataTemplate 自动关联 | 在 App.xaml 中定义 ViewModel-View 映射 |

### 视图导航流程

```
App启动 → MainWindow加载 → 导航到LoginView
    ↓
登录成功(非游客) → 导航到PatientSelectionView → 选择患者 → 导航到MainContainerView
    ↓
登录成功(游客) → 直接导航到MainContainerView（无当前患者）
    ↓
MainContainerView内部 → 通过导航栏切换子视图（测量/数据管理/报告/设置）
    ↓
退出登录 → 清理会话 → 导航回LoginView
```
