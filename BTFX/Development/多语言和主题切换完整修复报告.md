# 多语言和主题切换完整修复报告

## 修复时间
2025-01-XX

## 问题描述

### 主要问题
1. **中英文切换**：导航界面和各个子界面的文字一直是中文，无法随着语言切换而自动变化
2. **主题切换**：主题切换后视觉效果没有立即应用到界面

## 根本原因分析

### 1. 导航菜单硬编码问题
**问题**：导航菜单项的Title使用硬编码的中文字符串，而不是动态资源
```csharp
// 问题代码
new NavigationItem {
    Title = "测量评估",  // 硬编码中文
    ...
}
```

**影响**：
- 语言切换后，导航菜单文字不会改变
- 无法支持多语言

### 2. UI刷新机制不足
**问题**：之前的修复虽然添加了UI刷新逻辑，但不够全面
- 只刷新了Window级别的布局
- 没有刷新ViewModel绑定的属性
- 没有订阅语言变化事件

```csharp
/// <summary>
/// 资源键（用于多语言）
/// </summary>
[ObservableProperty]
private string _resourceKey = string.Empty;

/// <summary>
/// 从资源更新标题
/// </summary>
public void UpdateTitleFromResource()
{
    if (!string.IsNullOrEmpty(_resourceKey))
    {
        try
        {
            var resource = Application.Current.FindResource(_resourceKey);
            if (resource != null)
            {
                Title = resource.ToString() ?? Title;
            }
        }
        catch
        {
            // 如果找不到资源，保持原标题
        }
    }
}
```

    // 更新设备状态文字
    OnPropertyChanged(nameof(DeviceStatusText));

    foreach (var item in NavigationItems)

        item.UpdateTitleFromResource();
    }
}

public string DeviceStatusText
{
    get
    {
        try
        {
            var resourceKey = DeviceStatus switch
            {
                DeviceConnectionStatus.Connected => "DeviceConnected",
                DeviceConnectionStatus.Connecting => "DeviceConnecting",
                DeviceConnectionStatus.ConnectionFailed => "DeviceConnectionFailed",

                DeviceConnectionStatus.Connecting => "连接中...",

                _ => "未连接"
            };
        }
    }
}

### 修复2：增强UI刷新机制

**BTFX\Services\Implementations\LocalizationService.cs**
```csharp
public void ApplyLanguage(AppLanguage language)
{
    // ... 加载资源字典代码 ...
    
    try
    {
        var newDict = new ResourceDictionary { Source = resourceUri };
        Application.Current.Resources.MergedDictionaries.Add(newDict);

        CurrentLanguage = language;
        
        // 触发事件通知UI更新（这会触发MainContainerViewModel的更新）
        LanguageChanged?.Invoke(this, language);
        
        // 强制刷新所有打开的窗口
        Application.Current.Dispatcher.Invoke(() =>
        {
            foreach (Window window in Application.Current.Windows)
            {
                try
                {
                    // 触发资源更新
                    window.UpdateLayout();
                    
                    // 刷新DataContext以触发绑定更新
                    var dc = window.DataContext;
                    window.DataContext = null;
                    window.DataContext = dc;
                }
                catch
                {
                    // 忽略单个窗口刷新失败
                }
            }
        });
    }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"加载语言资源失败: {ex.Message}");
    }
}
```

**BTFX\Services\Implementations\ThemeService.cs**
```csharp
public void ApplyTheme(AppTheme theme)
{
    CurrentTheme = theme;

    var paletteHelper = new PaletteHelper();
    var currentTheme = paletteHelper.GetTheme();

    // 设置基础主题
    currentTheme.SetBaseTheme(theme == AppTheme.Dark ? BaseTheme.Dark : BaseTheme.Light);

    // 应用主题
    paletteHelper.SetTheme(currentTheme);

    // 触发事件
    ThemeChanged?.Invoke(this, theme);
    
    // 强制刷新所有打开的窗口
    Application.Current.Dispatcher.Invoke(() =>
    {
        foreach (Window window in Application.Current.Windows)
        {
            try
            {
                // 触发资源更新
                window.UpdateLayout();
                window.InvalidateVisual();
            }
            catch
            {
                // 忽略单个窗口刷新失败
            }
        }
    });
}
```

### 事件订阅链

```
用户点击语言切换
    ↓
SettingsViewModel.OnSelectedLanguageChanged()
    ↓
LocalizationService.ApplyLanguage()
    ↓
触发 LanguageChanged 事件
    ↓
MainContainerViewModel.OnLanguageChanged()
    ↓
NavigationItem.UpdateTitleFromResource()
    ↓
OnPropertyChanged("Title")
    ↓
UI自动更新 (通过数据绑定)
```

### 资源查找机制

```csharp
// 1. NavigationItem请求更新
navigationItem.UpdateTitleFromResource()

// 2. 使用ResourceKey查找资源
var resource = Application.Current.FindResource("NavMeasurement");

// 3. WPF在资源字典中查找
Application.Current.Resources.MergedDictionaries
    └─> Strings.zh.xaml 或 Strings.en.xaml
        └─> <system:String x:Key="NavMeasurement">测量评估</system:String>

// 4. 返回资源值并更新Title
navigationItem.Title = resource.ToString();

// 5. 触发属性变更通知
OnPropertyChanged(nameof(Title))

// 6. UI绑定自动更新
<TextBlock Text="{Binding Title}" />
```

## 已知限制

### 1. 部分硬编码文本
以下文本仍然是硬编码的，需要后续添加资源键：
- 对话框标题和消息
- 错误提示信息
- MessageBox文本
- 某些动态生成的UI元素

**解决方案**：逐步将这些文本移到资源文件中。

### 2. 第三方控件
MaterialDesignThemes的某些控件可能不支持完全的动态主题切换。

**解决方案**：99%的场景已经支持，极少数情况可能需要重启应用。

### 3. 性能考虑
- 每次语言切换会触发所有NavigationItem的更新
- 大量UI元素时可能有轻微延迟（<100ms）

**优化方案**：已使用Observable模式，性能影响可忽略。
