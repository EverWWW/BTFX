## 4.1 服务层架构

### 服务接口设计原则

1. 所有服务定义接口（Interface）
2. 接口放在 `Services/Interfaces/` 目录
3. 实现放在 `Services/Implementations/` 目录
4. 通过 Microsoft.Extensions.DependencyInjection 注入
5. 异步方法使用 `async/await`

### 服务清单

| 服务接口 | 实现类 | 生命周期 | 说明 |
|---------|-------|---------|------|
| INavigationService | NavigationService | Singleton | 导航服务 |

### 删除权限检查逻辑

```csharp
public async Task<bool> CanDeletePatientAsync(int patientId, int operatorId)
{
    var patient = await GetPatientByIdAsync(patientId);
    if (patient == null) return false;

    var user = await _userService.GetUserByIdAsync(operatorId);
    if (user == null) return false;

    // 管理员可删除任意患者
    if (user.Role == UserRole.Administrator) return true;

    // 操作员只能删除自己创建的患者
    if (user.Role == UserRole.Operator)
        return patient.CreatedBy == operatorId;

    return false;
}
```

## 4.6 报告服务 (IReportService)

### 接口定义

| 方法 | 返回值 | 说明 |
|-----|-------|------|
| GetReportsAsync(filter, page, pageSize) | PagedResult<Report> | 分页查询报告 |
| GetReportByIdAsync(id) | Report? | 按ID查询 |
| GetReportByMeasurementIdAsync(measurementId) | Report? | 按测量记录ID查询报告 |
| HasReportForMeasurementAsync(measurementId) | bool | 检查测量记录是否已有报告 |
| GenerateReportAsync(measurementId, opinion, overwrite) | Result<Report> | 生成报告（支持覆盖） |
| UpdateReportAsync(report) | Result | 更新报告 |
| DeleteReportAsync(id) | Result | 删除报告 |
| ExportToPdfAsync(reportId, filePath) | Result | 导出PDF |
| PrintReportAsync(reportId) | Result | 打印报告 |

### 报告生成流程

```
1. 检查测量数据是否存在
2. 检查是否已有报告
   - 若有报告且 overwrite=false，返回错误
   - 若有报告且 overwrite=true，更新现有报告
   - 若无报告，创建新报告
3. 获取患者信息
4. 获取单位信息
5. 填充报告模板
6. 添加医生意见
7. 生成报告文件
8. 保存/更新报告记录
```

### 报告覆盖逻辑

- 保留原报告编号
- 更新报告内容和医生意见
- 更新 UpdatedAt 时间戳
- 重置报告状态为 Draft

## 4.14 服务注册配置

### 在 App.xaml.cs 中注册服务

```csharp
private void ConfigureServices(IServiceCollection services)
{
    // ========== Singleton 服务 ==========
    // 导航服务
    services.AddSingleton<INavigationService, NavigationService>();

    // 会话服务
    services.AddSingleton<ISessionService, SessionService>();

    // 设置服务
    services.AddSingleton<ISettingsService, SettingsService>();

    // 多语言服务
    services.AddSingleton<ILocalizationService, LocalizationService>();

    // 主题服务
    services.AddSingleton<IThemeService, ThemeService>();

    // 认证服务
    services.AddSingleton<IAuthenticationService, AuthenticationService>();

    // 备份服务
    services.AddSingleton<IBackupService, BackupService>();

    // ========== Transient 服务 ==========
    // 患者服务
    services.AddTransient<IPatientService, PatientService>();

    // 测量服务
    services.AddTransient<IMeasurementService, MeasurementService>();

    // 报告服务
    services.AddTransient<IReportService, ReportService>();

    // 用户管理服务
    services.AddTransient<IUserService, UserService>();

    // 导入导出服务
    services.AddTransient<IExportImportService, ExportImportService>();

    // 科室服务
    services.AddTransient<IDepartmentService, DepartmentService>();

    // ========== ViewModel 注册 ==========
    services.AddTransient<MainWindowViewModel>();
    services.AddTransient<LoginViewModel>();
    services.AddTransient<PatientSelectionViewModel>();
    services.AddTransient<MainContainerViewModel>();
    services.AddTransient<MeasurementViewModel>();
    services.AddTransient<DataManagementViewModel>();
    services.AddTransient<ReportViewModel>();
    services.AddTransient<SettingsViewModel>();

    // ========== View 注册 ==========
    services.AddTransient<MainWindow>();
    services.AddTransient<LoginView>();
    services.AddTransient<PatientSelectionView>();
    services.AddTransient<MainContainerView>();
    services.AddTransient<MeasurementView>();
    services.AddTransient<DataManagementView>();
    services.AddTransient<ReportView>();
    services.AddTransient<SettingsView>();

    // ========== 对话框注册 ==========
    services.AddTransient<PatientEditDialog>();
    services.AddTransient<UserManageDialog>();
    services.AddTransient<PasswordChangeDialog>();
    services.AddTransient<BackupRestoreDialog>();
    services.AddTransient<ConfirmDialog>();
    services.AddTransient<AboutDialog>();
}
```

### 服务获取方式

在 ViewModel 中通过构造函数注入：

```csharp
public partial class PatientSelectionViewModel : ObservableObject
{
    private readonly IPatientService _patientService;
    private readonly INavigationService _navigationService;
    private readonly ISessionService _sessionService;

    public PatientSelectionViewModel(
        IPatientService patientService,
        INavigationService navigationService,
        ISessionService sessionService)
    {
        _patientService = patientService;
        _navigationService = navigationService;
        _sessionService = sessionService;

        LoadPatientsAsync();
    }
}
```

### 在非DI场景获取服务

如果需要在非构造函数场景获取服务，可以通过 `App.Services`：

```csharp
var patientService = App.Services.GetRequiredService<IPatientService>();
```
