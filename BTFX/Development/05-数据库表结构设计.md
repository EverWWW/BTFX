# 数据库表结构设计

**数据库类型**: SQLite  
**文件位置**: Data/Database/BTFX.db  
**访问方式**: ToolHelper.Database (SqlSugar ORM)
**数据库版本**: 1

---

## 表结构总览

| 表名 | 说明 |
|-----|------|
| Users | 用户表 |
| Patients | 患者表 |
| Departments | 科室表 |
| MeasurementRecords | 测量记录表 |
| GaitParameters | 步态参数表 |
| Reports | 报告表 |
| SystemSettings | 系统设置表 |
| OperationLogs | 操作日志表 |

---

## 1. Users 用户表

### 字段定义

| 字段名 | 类型 | 约束 | 说明 |
|-------|------|------|------|
| Id | INTEGER | PRIMARY KEY AUTOINCREMENT | 用户ID |
| Username | TEXT | NOT NULL UNIQUE | 账号 (3-20字符) |
| PasswordHash | TEXT | NOT NULL | 密码哈希 |
| PasswordSalt | TEXT | NOT NULL | 密码盐值 |
| Name | TEXT | NOT NULL | 姓名 (2-30字符) |
| Phone | TEXT | NOT NULL | 电话 (3-12位) |
| Role | INTEGER | NOT NULL DEFAULT 1 | 角色 (0-管理员,1-操作员,2-游客) |
| DepartmentId | INTEGER | NULL | 科室ID (外键) |
| IsEnabled | INTEGER | NOT NULL DEFAULT 1 | 是否启用 (0-禁用,1-启用) |
| IsBuiltIn | INTEGER | NOT NULL DEFAULT 0 | 是否内置账号 (不可删除) |
| LastLoginAt | TEXT | NULL | 最后登录时间 |
| CreatedAt | TEXT | NOT NULL | 创建时间 |
| UpdatedAt | TEXT | NOT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 说明 |
|-------|------|------|
| IX_Users_Username | Username | 账号唯一索引 |
| IX_Users_Phone | Phone | 电话索引 |

### 内置数据

| Username | Name | Role | IsBuiltIn | 默认密码 |
|----------|------|------|-----------|---------|
| admin | 管理员 | 0 (Administrator) | 1 | 688626 |
| user | 操作员 | 1 (Operator) | 1 | 688626 |

## 2. Patients 患者表

### 字段定义

| 字段名 | 类型 | 约束 | 说明 |
|-------|------|------|------|
| Id | INTEGER | PRIMARY KEY AUTOINCREMENT | 患者ID |
| Name | TEXT | NOT NULL | 姓名 (2-30字符) |
| Gender | INTEGER | NOT NULL DEFAULT 0 | 性别 (0-男,1-女) |
| BirthDate | TEXT | NULL | 出生日期 |
| Phone | TEXT | NOT NULL | 电话 (3-12位,需去重) |
| IdNumber | TEXT | NULL | 证件号 (最长18位) |
| Height | REAL | NULL | 身高cm (1-300) |
| Weight | REAL | NULL | 体重kg (1-500) |
| Address | TEXT | NULL | 地址 |
| MedicalHistory | TEXT | NULL | 病史 |
| Remark | TEXT | NULL | 备注 |
| Status | INTEGER | NOT NULL DEFAULT 0 | 状态 (0-正常,1-已删除) |
| CreatedBy | INTEGER | NOT NULL | 创建人ID |
| CreatedAt | TEXT | NOT NULL | 创建时间 |
| UpdatedAt | TEXT | NOT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 说明 |
|-------|------|------|
| IX_Patients_Name | Name | 姓名索引 |
| IX_Patients_Phone | Phone | 电话索引 |
| IX_Patients_IdNumber | IdNumber | 证件号索引 |
| IX_Patients_Status | Status | 状态索引 |

### 约束说明

- 电话在未删除记录中唯一
- 证件号18位时需验证身份证规则（应用层验证）

## 6. Reports 报告表

### 字段定义

| 字段名 | 类型 | 约束 | 说明 |
|-------|------|------|------|
| Id | INTEGER | PRIMARY KEY AUTOINCREMENT | 报告ID |
| MeasurementId | INTEGER | NOT NULL | 测量记录ID (外键) |
| PatientId | INTEGER | NOT NULL | 患者ID (外键) |

| DoctorOpinion | TEXT | NULL | 医生意见 |

## 初始化脚本要点

### 数据库创建顺序

1. Departments（无外键依赖）
2. Users（依赖Departments）
3. Patients（依赖Users）
4. MeasurementRecords（依赖Patients, Users）
5. GaitParameters（依赖MeasurementRecords）
6. Reports（依赖MeasurementRecords, Patients, Users）
7. SystemSettings（无外键依赖）
8. OperationLogs（无外键依赖）

### 初始数据

1. 创建内置用户 admin 和 user
2. 插入默认系统设置

### 数据库版本管理

| 字段 | 说明 |
|-----|------|
| 版本表 | 或使用 SQLite PRAGMA user_version |
| 升级脚本 | 按版本号执行迁移脚本 |

## 数据类型说明

### SQLite 类型映射

| C# 类型 | SQLite 类型 | 说明 |
|--------|-------------|------|
| int | INTEGER | 整数 |
| long | INTEGER | 长整数 |
| double/float | REAL | 浮点数 |
| string | TEXT | 文本 |
| bool | INTEGER | 0/1 |
| DateTime | TEXT | ISO8601格式字符串 |
| byte[] | BLOB | 二进制 |

### 日期时间格式

统一使用 ISO8601 格式：`yyyy-MM-ddTHH:mm:ss`

## 待确认问题（已全部确认）

> **Q-5.1**: 步态参数的具体字段？是使用独立字段还是JSON存储？ - **使用独立字段存储，具体字段需与算法团队确认**
>
> **Q-5.2**: 视频文件是否需要在数据库中存储元数据（分辨率、帧率等）？ - **不需要**
>
> **Q-5.3**: 是否需要数据加密存储？哪些字段需要加密？ - **仅加密用户密码，证件号不加密**
>
> **Q-5.4**: 操作日志保留策略：保留多长时间？是否需要自动清理？ - **保留90天，启动时自动清理**
>
> **Q-5.5**: 是否需要支持数据导出后再导入（数据迁移场景）？ - **是的，支持数据导出导入，方便数据迁移**
>
> **Q-5.6**: 视频文件存储路径格式？ - **使用相对路径，便于数据迁移**
>
> **Q-5.7**: 数据库版本号起始值？ - **从1开始**
