
## 附件：WPF 调用示例 / Python 输出示例

### 1. WPF 调用示例

WPF 上位机通过 `Process` 启动算法 exe，并传入任务配置文件路径；算法模块运行过程中通过标准输出返回任务状态，上位机逐行读取并解析。

示例代码如下：

```csharp
using System;
using System.Diagnostics;
using System.Text.Json;
using System.Threading.Tasks;

public class GaitTaskRunner
{
    public async Task<int> RunAsync(string exePath, string configPath)
    {
        var startInfo = new ProcessStartInfo
        {
            FileName = exePath,
            Arguments = $"--config \"{configPath}\"",
            UseShellExecute = false,
            RedirectStandardOutput = true,
            RedirectStandardError = true,
            CreateNoWindow = true
        };

        using var process = new Process();
        process.StartInfo = startInfo;

        process.OutputDataReceived += (s, e) =>
        {
            if (string.IsNullOrWhiteSpace(e.Data))
                return;

            try
            {
                var status = JsonSerializer.Deserialize<TaskStatusMessage>(e.Data);
                if (status != null)
                {
                    Console.WriteLine($"[{status.task_status}] {status.progress}% - {status.message}");
                    // 这里可更新界面进度条、状态文本
                }
            }
            catch
            {
                // 非 JSON 输出可作为普通日志处理
            }
        };

        process.ErrorDataReceived += (s, e) =>
        {
            if (!string.IsNullOrWhiteSpace(e.Data))
            {
                Console.WriteLine("ERR: " + e.Data);
            }
        };

        process.Start();
        process.BeginOutputReadLine();
        process.BeginErrorReadLine();

        await process.WaitForExitAsync();
        return process.ExitCode;
    }
}

public class TaskStatusMessage
{
    public string type { get; set; }
    public string request_id { get; set; }
    public string task_status { get; set; }
    public int progress { get; set; }
    public string message { get; set; }
    public int? error_code { get; set; }
}
```

### 2. Python 输出示例

Python 算法模块接收 `--config` 参数，读取任务配置文件，在执行过程中按协议向 stdout 输出单行 JSON 状态消息，供上位机实时解析。

示例代码如下：

```python
import argparse
import json
import sys
from pathlib import Path

def emit_status(request_id, task_status, progress, message, error_code=None):
    payload = {
        "type": "status",
        "request_id": request_id,
        "task_status": task_status,
        "progress": progress,
        "message": message
    }
    if error_code is not None:
        payload["error_code"] = error_code
    print(json.dumps(payload, ensure_ascii=False), flush=True)

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--config", required=True)
    args = parser.parse_args()

    config_path = Path(args.config)
    if not config_path.exists():
        emit_status("UNKNOWN", "failed", 100, "config file not found", 1)
        return 1

    config = json.loads(config_path.read_text(encoding="utf-8"))
    request_id = config["request_id"]

    emit_status(request_id, "pending", 0, "task accepted")
    emit_status(request_id, "running", 10, "checking input files")
    emit_status(request_id, "running", 35, "detecting keypoints")
    emit_status(request_id, "running", 60, "detecting gait events")
    emit_status(request_id, "running", 80, "computing gait parameters")
    emit_status(request_id, "completed", 100, "analysis completed")

    return 0

if __name__ == "__main__":
    sys.exit(main())
```

### 3. 状态消息示例

算法模块在执行过程中输出的标准状态消息示例如下：

任务开始：

```json
{"type":"status","request_id":"GAIT_20260324_001","task_status":"pending","progress":0,"message":"task accepted"}
```

执行中：

```json
{"type":"status","request_id":"GAIT_20260324_001","task_status":"running","progress":35,"message":"detecting keypoints"}
```

分析完成：

```json
{"type":"status","request_id":"GAIT_20260324_001","task_status":"completed","progress":100,"message":"analysis completed"}
```

分析失败：

```json
{"type":"status","request_id":"GAIT_20260324_001","task_status":"failed","progress":100,"message":"video file not found","error_code":2}
```
