## 1.4 WPF+MVVM 框架配置

### 使用 CommunityToolkit.Mvvm

所有 ViewModel 直接继承 `ObservableObject`，使用 `[ObservableProperty]` 和 `[RelayCommand]` 特性简化代码：

```csharp
using CommunityToolkit.Mvvm.ComponentModel;
using CommunityToolkit.Mvvm.Input;

public partial class LoginViewModel : ObservableObject
{
    [ObservableProperty]
    private string _username = string.Empty;

    [ObservableProperty]
    private string _password = string.Empty;

    [ObservableProperty]
    private bool _isLoggingIn;

    [RelayCommand(CanExecute = nameof(CanLogin))]
    private async Task LoginAsync()
    {
        IsLoggingIn = true;
        // 登录逻辑
        IsLoggingIn = false;
    }

    private bool CanLogin() => !string.IsNullOrEmpty(Username) && !string.IsNullOrEmpty(Password);
}
```

### 依赖注入配置 (Microsoft.Extensions.DependencyInjection)

在 `App.xaml.cs` 中配置服务容器：

```csharp
public partial class App : Application
{
    public static IServiceProvider Services { get; private set; } = null!;

    protected override void OnStartup(StartupEventArgs e)
    {
        base.OnStartup(e);

        var services = new ServiceCollection();
        ConfigureServices(services);
        Services = services.BuildServiceProvider();

        // 显示主窗口
        var mainWindow = Services.GetRequiredService<MainWindow>();
        mainWindow.Show();
    }

    private void ConfigureServices(IServiceCollection services)
    {
        // 注册服务（Singleton）
        services.AddSingleton<INavigationService, NavigationService>();
        services.AddSingleton<ISessionService, SessionService>();
        services.AddSingleton<ISettingsService, SettingsService>();
        services.AddSingleton<ILocalizationService, LocalizationService>();
        services.AddSingleton<IThemeService, ThemeService>();
        services.AddSingleton<IAuthenticationService, AuthenticationService>();
        services.AddSingleton<IBackupService, BackupService>();

        // 注册服务（Transient）
        services.AddTransient<IPatientService, PatientService>();
        services.AddTransient<IMeasurementService, MeasurementService>();
        services.AddTransient<IReportService, ReportService>();
        services.AddTransient<IUserService, UserService>();
        services.AddTransient<IExportImportService, ExportImportService>();
        services.AddTransient<IDepartmentService, DepartmentService>();

        // 注册 ViewModels
        services.AddTransient<MainWindowViewModel>();
        services.AddTransient<LoginViewModel>();
        services.AddTransient<PatientSelectionViewModel>();
        services.AddTransient<MainContainerViewModel>();
        services.AddTransient<MeasurementViewModel>();
        services.AddTransient<DataManagementViewModel>();
        services.AddTransient<ReportViewModel>();
        services.AddTransient<SettingsViewModel>();

        // 注册 Views
        services.AddTransient<MainWindow>();
        services.AddTransient<LoginView>();
        services.AddTransient<PatientSelectionView>();
        // ... 其他视图
    }
}
```

### ViewModel 获取服务

在 ViewModel 中通过构造函数注入获取服务：

```csharp
public partial class PatientSelectionViewModel : ObservableObject
{
    private readonly IPatientService _patientService;
    private readonly INavigationService _navigationService;
    private readonly ISessionService _sessionService;

    public PatientSelectionViewModel(
        IPatientService patientService,
        INavigationService navigationService,
        ISessionService sessionService)
    {
        _patientService = patientService;
        _navigationService = navigationService;
        _sessionService = sessionService;
    }
}
```

### App.xaml.cs 主要方法

1. **OnStartup()**: 
   - 单实例检测（Mutex）
   - 全局异常处理注册
   - 配置依赖注入
   - 加载配置文件
   - 应用语言和主题设置
   - 初始化数据库和必要文件夹
   - 创建并显示主窗口

2. **OnExit()**: 
   - 释放Mutex
   - 保存配置
   - 清理资源

### 单实例检测

- 使用 Mutex 实现
- Mutex 名称：`Global\BTFX_SingleInstance_Mutex`
- 重复启动时显示提示："程序已运行，请勿重复启动。"

### 全局异常处理

- 注册 `DispatcherUnhandledException`
- 注册 `AppDomain.CurrentDomain.UnhandledException`
- 注册 `TaskScheduler.UnobservedTaskException`
- 异常时记录日志并显示友好提示

### 自定义导航服务 (INavigationService)

```csharp
public interface INavigationService
{
    object? CurrentView { get; }
    string CurrentViewKey { get; }

    void NavigateTo<TViewModel>() where TViewModel : ObservableObject;
    void NavigateTo(string viewKey);
    bool CanGoBack { get; }
    void GoBack();
}

public class NavigationService : ObservableObject, INavigationService
{
    private readonly IServiceProvider _serviceProvider;
    private readonly Dictionary<Type, Type> _viewModelToViewMap = new();

    [ObservableProperty]
    private object? _currentView;

    [ObservableProperty]
    private string _currentViewKey = string.Empty;

    public NavigationService(IServiceProvider serviceProvider)
    {
        _serviceProvider = serviceProvider;
        RegisterViewMappings();
    }

    private void RegisterViewMappings()
    {
        _viewModelToViewMap[typeof(LoginViewModel)] = typeof(LoginView);
        _viewModelToViewMap[typeof(PatientSelectionViewModel)] = typeof(PatientSelectionView);
        _viewModelToViewMap[typeof(MainContainerViewModel)] = typeof(MainContainerView);
        // ... 其他映射
    }

    public void NavigateTo<TViewModel>() where TViewModel : ObservableObject
    {
        var viewModel = _serviceProvider.GetRequiredService<TViewModel>();
        var viewType = _viewModelToViewMap[typeof(TViewModel)];
        var view = (FrameworkElement)_serviceProvider.GetRequiredService(viewType);
        view.DataContext = viewModel;
        CurrentView = view;
        CurrentViewKey = typeof(TViewModel).Name;
    }
}
```

## 1.5 Material Design 集成

### NuGet 包

- `MaterialDesignThemes` (最新稳定版)
- `MaterialDesignColors`

### App.xaml 配置要求

1. 引入 Material Design 资源字典
2. 配置默认主题：Light
3. 配置默认主色调：Blue (#2196F3)
4. 配置默认副色调：LightBlue (#03A9F4)
5. 配置全局字体：阿里巴巴普惠体

### 字体配置要求

**字体名称**: 阿里巴巴普惠体 (Alibaba PuHuiTi)

**字体文件**:
- `Resources/Fonts/AlibabaPuHuiTi-Regular.ttf` - 常规字重
- `Resources/Fonts/AlibabaPuHuiTi-Medium.ttf` - 中等字重
- `Resources/Fonts/AlibabaPuHuiTi-Bold.ttf` - 粗体字重

**配置方式**:
1. 将字体文件添加到项目 `Resources/Fonts/` 目录
2. 设置字体文件的生成操作为 `Resource`
3. 在 App.xaml 中定义全局字体样式
4. 设置 Window 和所有控件的默认字体

**字体引用格式**:
```xaml
FontFamily="pack://application:,,,/Resources/Fonts/#Alibaba PuHuiTi"
```

**字重使用规范**:
| 场景 | 字重 | 字号参考 |
|-----|------|---------|
| 标题 | Bold | 18-24px |
| 副标题 | Medium | 14-16px |
| 正文 | Regular | 12-14px |
| 辅助文字 | Regular | 10-12px |

### 主题切换服务 (IThemeService)

| 方法/属性 | 说明 |
|----------|------|
| CurrentTheme | 当前主题属性 |
| ApplyTheme(AppTheme) | 应用指定主题 |
| ToggleTheme() | 切换亮色/深色主题 |

**要求**: 主题切换实时生效，无需重启
