【English Description】

1. Introduction
The Gait Intelligent Analysis System (BTFX) is a professional medical auxiliary 
software designed for analyzing and evaluating patient gait parameters, providing 
data support for clinical diagnosis and rehabilitation assessment.

2. System Requirements
- Operating System: Windows 10 (1809+) or Windows 11
- Processor: x64 architecture, Intel Core i5 or equivalent
- Memory: 8GB RAM or above
- Disk Space: 500MB available space
- Display: 1920x1080 resolution or higher

3. Key Features
- Gait data collection and analysis
- Patient information management
- Measurement record management
- Report generation and printing
- Data import and export
- Multi-language support (Chinese/English)

4. Installation
1. Run the installer
2. Follow the installation wizard
3. Choose installation path
4. Launch from desktop or Start menu after installation

5. Default Login Credentials
- Administrator: admin
- Default Password: admin123

6. Technical Support
For assistance, please contact BTFX Team technical support.
