### ?? 新增功能（v2.0）

#### 1. YamlHelper ?

**已实现功能**:
- ? YAML 序列化/反序列化
- ? 文件异步IO操作
- ? YAML 验证和格式化
- ? CamelCase 命名约定
- ? 完整的错误处理和日志

**代码示例**:
```csharp
var yamlHelper = new YamlHelper();

// 序列化
var config = new AppConfig { AppName = "MyApp", Version = "1.0" };
var yaml = yamlHelper.Serialize(config);

// 反序列化
var loaded = yamlHelper.Deserialize<AppConfig>(yaml);

// 文件操作
await yamlHelper.SerializeToFileAsync("config.yaml", config);
var fromFile = await yamlHelper.DeserializeFromFileAsync<AppConfig>("config.yaml");
```

**核心特性**:
- 基于 YamlDotNet 实现
- 支持复杂对象结构
- 自动命名转换
- 完整的日志记录

#### 2. ExcelHelper ?

**已实现功能**:
- ? Excel 读写（.xls 和 .xlsx）
- ? 流式读取大文件
- ? 自动类型转换
- ? 标题行样式支持
- ? 自动列宽调整
- ? 追加写入模式
- ? 日期格式处理

**代码示例**:
```csharp
var excelHelper = new ExcelHelper<Person>();

// 写入Excel
var data = new List<Person> { /* ... */ };
await excelHelper.WriteAsync("data.xlsx", data);

// 读取Excel
var result = await excelHelper.ReadAsync("data.xlsx");

// 流式读取
await foreach (var person in excelHelper.ReadStreamAsync("large.xlsx"))
{
    ProcessPerson(person);
}
```

**核心特性**:
- 基于 NPOI 实现
- 支持 .xls 和 .xlsx 格式
- 自动样式美化（标题行灰色背景、居中）
- 智能类型转换
- 完整的异常处理

#### 3. PdfHelper ?

**已实现功能**:
- ? 简单文本PDF生成
- ? 数据表格PDF生成
- ? 完整报表PDF生成（含统计信息）
- ? 自动分页
- ? 页眉页脚
- ? 页码支持
- ? 多种页面尺寸（A3, A4, A5等）
- ? 可配置页边距

**代码示例**:
```csharp
var pdfHelper = new PdfHelper();

// 生成文本PDF
await pdfHelper.GenerateTextPdfAsync("document.pdf", "内容文本");

// 生成表格PDF
await pdfHelper.GenerateTablePdfAsync("table.pdf", dataList, "数据表");

// 生成完整报表
var summary = new Dictionary<string, string>
{
    { "总计", "100" },
    { "平均值", "50" }
};
await pdfHelper.GenerateReportPdfAsync("report.pdf", dataList, "统计报表", summary);
```

**核心特性**:
- 基于 QuestPDF 实现（Community License）
- 专业的报表布局
- 自动表格样式（斑马纹、边框）
- 灵活的页面配置
- 完整的中文支持
